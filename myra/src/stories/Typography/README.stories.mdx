# Typography

`npm install --save myra-ui`

#### Usage
`import { Typography } from 'myra-ui';`

### Variants

<ul>
<li><h3><code>h1: 2rem</code></h3></li>
<li><h3><code>h2: 2rem</code></h3></li>
<li><h3><code>h3: 1rem</code></h3></li>
<li><h3><code>body1: 18px</code></h3></li>
</ul>

`{ h1: 'h1', h2: 'h2', h3: 'h3', h4: 'h4', h5: 'h5', h6: 'h6', subtitle1: 'h6', subtitle2: 'h6', body1: 'p', body2: 'p'}`

#### Example
`<Typography variant="body1" gutterBottom>Some text...</Typography>`
<Typography variant="body1" gutterBottom>
        body1. Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quos blanditiis tenetur
        unde suscipit, quam beatae rerum inventore consectetur, neque doloribus, cupiditate numquam
        dignissimos laborum fugiat deleniti? Eum quasi quidem quibusdam.
      </Typography>`
